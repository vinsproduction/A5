
	
	================ DEBUG MODE ===============	

	Дебаг режим позволяет просматривать ошибки. Это режим необходим только для разработчка!
	
	Для его включения добавьте к урлу ?debug=1
	
	Дефолтный логин и пароль:
	
	debug_login		= DebugMode
	debug_password 	= DebugMode_3000
	
	Переназначить дебаг-конфиг можно в application/config/main.ini	
	

	=============== CONSOLE ===================

	
	Для открытия жми [ctrl + ~]
	
	(!) Консоль можно просматривать только в DEBUG режиме
