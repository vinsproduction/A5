Как установить административную часть в систему.

1) Использовать скрипт a5.php для инсталляции.
   Пример: php a5.php install cms into /path/to/application/folder
 
   После этого, если вы хотите изменить расположение некоторых или всех папок или их имена вы
   должны определить соотвествующие константы (или указать их в конфиг-файлах). Описание соотвествующих констант
   вы можете найти в самом начале файла plugins/cms/controllers/include/api.php

2) В файле application/controllers/application.php подключить код системы администрирования.
   Пример: require("cms/include/setup.php");

3) При необходимости перед подключением файла в пункте 2 можно определить константу
   CMS_MEDIA_DIR которая должна содержать абсолютный и обработанный функцией normalize_path путь
   к папкам где будут храниться статичные данные из системы администрирования (картинки, флэш, файлы),
   путь должен находится внутри PUBLIC_DIR.
   При необходимости, абсолютные пути к этим папкам можно будет получить из констант:
   CMS_MEDIA_IMAGE_DIR, CMS_MEDIA_FLASH_DIR, CMS_MEDIA_FILES_DIR
      
4) Настроить в urlmapping.ini особые мапы нужные для более красивых урл
   Пример: чтобы работал более простой адрес: http://www.yoursite.com/admin
   нужно добавить в начало следующий мап:
   [admin => cms/index]
   
   Чтобы объекты типа текстовой страницы отображались более красиво
   [text/@id => documents#view]

5) Зайти по-адресу: http://www.yoursite.com/cms/index
   На предложенный запрос создания базы данных - ответить утвердительно и дождаться завершения
   операции создания объектов.

6) Для первоначального входа в систему управления используйте пароль и логин: dev
   Не забудьте сменить пароль после входа.
